a rock paper scissors realtime game

[Demo](https://rock-paper-scissors-rt.herokuapp.com/)

## server side
- Express http server 
- SocketIo server
- Heroku
- Github actions
## client side (the game)
- typescript
- custom game engine
- bundle with Webpack
- socket-io-client
- simple UI
